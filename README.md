# Challenge Game App

Challenge Game is a fun and interactive app where users can participate in different challenges, complete tasks, and track their progress.

The main goal of the app is to motivate users to step out of their comfort zone, improve their skills, and compete with friends or themselves. Each challenge has clear rules and objectives, making the experience simple and engaging.

# Users can:

- Join different challenges

- Complete tasks and earn points

- Track their progress

- Compare results with others

The app was built to be easy to use, fast, and motivating. It focuses on creating a dynamic and competitive environment while keeping the experience enjoyable.
